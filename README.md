# Image Forgery Detection System

## Overview

This project uses the Meso4 Convolutional Neural Network (CNN) architecture to detect forged or manipulated images.

The system allows users to upload an image through a graphical interface and predicts whether the image is real or fake.

## Features

- Image Upload using Tkinter
- Image Preprocessing
- CNN-based Meso4 Model
- Real/Fake Classification
- Prediction Visualization
- Deep Learning based Detection

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- PIL
- Tkinter
- Matplotlib

## Project Workflow

1. Upload Image
2. Resize and Normalize
3. Process using Meso4 CNN
4. Predict Real or Fake
5. Display Result
